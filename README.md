# Угадай слово (GuessWord)

WinForms-приложение — игра «Угадай слово» по правилам виселицы.

## Архитектура

Проект построен по принципам ООП — три основных класса:

### Player (Игрок)
Хранит данные пользователя и историю игр.

| Поле/Свойство | Тип | Описание |
|---|---|---|
| `Login` | `string` | Логин игрока |
| `Password` | `string` | Пароль |
| `Games` | `List<Game>` | Список сыгранных игр |
| `GamesPlayed()` | `int` | Количество сыгранных игр |
| `GamesWon()` | `int` | Количество выигранных игр |

### Game (Игра)
Хранит состояние текущей игры: слово, открытые буквы, попытки.

| Поле/Свойство | Тип | Описание |
|---|---|---|
| `Word` | `string` | Загаданное слово |
| `Revealed` | `char[]` | Открытые буквы (`_` = скрытая) |
| `AttemptsLeft` | `int` | Оставшиеся попытки (6 на старте) |
| `TriedLetters` | `List<char>` | Уже названные буквы |
| `IsWon` | `bool` | Победа? |
| `IsOver` | `bool` | Игра окончена? |

Статический список слов: зима, источник, пенал, молоток, клавиатура, монитор, программа, калькулятор, велосипед, путешествие, шоколад, библиотека.

### Logic (Логика)
Правила игры и обработка ходов.

| Метод | Возвращает | Описание |
|---|---|---|
| `StartNewGame()` | `Game` | Начать новую игру (случайное слово) |
| `GuessLetter(char)` | `bool` | Угадать букву. `true` — буква есть в слове |
| `GuessWord(string)` | `bool` | Угадать слово целиком. Неверно = −1 попытка |
| `Hint()` | `char` | Подсказка: открыть первую скрытую букву (−1 попытка) |

## Правила игры

1. **Логин** — при запуске нужно ввести логин и пароль
2. **Загаданное слово** отображается как `_ _ _ _` (скрытые буквы)
3. **6 попыток** — каждая неверная буква или слово снимает 1 попытку
4. **Угадать букву** — ввести 1 букву и нажать «Буква» или Enter
5. **Угадать слово** — ввести слово целиком и нажать «Слово»
6. **Подсказка** — открывает первую скрытую букву, но стоит 1 попытку
7. Повторные буквы не снимают попытку, но и не считаются
8. Победа — все буквы открыты. Поражение — 0 попыток

## Интерфейс

- **Экран логина** — поля для логина и пароля, кнопка «Войти»
- **Игровое поле**:
  - Загаданное слово (крупный шрифт)
  - Счётчик попыток (цвет: 🟢 > 4, 🟠 3–4, 🔴 ≤ 2)
  - Список названных букв
  - Поле ввода буквы + кнопка «Буква»
  - Поле ввода слова + кнопка «Слово»
  - Кнопка «Подсказка» (−1)
  - Кнопка «Новая игра»
